# Reinforcement Learning Assignments
This repository contains several Reinforcement Learning algorithms implementations developed during the Reinforcement Learning course (A.Y. 2025/2026) of the MSc in Artificial Intelligence and Robotics.

The goal of these assignments was to implement classical and modern RL algorithms from scratch and apply them to different environments using Gymnasium.

## Implemented Algorithms

Dynamic Programming

Policy Iteration

Value Iteration

Model-Based Control

iLQR (Iterative Linear Quadratic Regulator)

Temporal Difference Learning

SARSA(λ)

Q-Learning with eligibility traces

Function Approximation

Linear Q-learning with RBF feature encoding

Deep Reinforcement Learning

Deep RL algorithm for the CarRacing-v2 environment (PyTorch implementation)


## Environments
The algorithms were tested on several reinforcement learning environments:

FrozenLake (custom environment)

Taxi

MountainCar

Pendulum

CarRacing-v2

All environments are based on Gymnasium.
